# ClassTimer - 课程倒计时

## 项目简介

ClassCountdown（课程倒计时）是一款专为学生设计的时间管理工具应用。通过导入课程表，应用会自动显示距离下一节课程（上课或下课）的剩余时间，帮助学生合理安排学习和休息时间。

## 核心功能

- **实时倒计时**：显示距离下一节课（上课/下课）的精确时间
- **课程表导入**：支持通过JSON文件或文本方式导入课程表
- **时间偏移设置**：可自定义铃声提前或延后的时间（秒）
- **智能计算**：内置时间偏移计算器，帮助用户快速设定提醒时间
- **双屏界面**：主屏显示倒计时信息，设置屏进行个性化配置

## 使用方法

### 1. 导入课程表

在设置界面可通过两种方式导入课程表：

#### 文件导入
准备一个JSON格式的课程表文件，格式如下：
```json
{
  "schedule": [
    {
      "day": 1,
      "classes": [
        {"name": "数学", "start": "08:00", "end": "08:45"},
        {"name": "语文", "start": "09:00", "end": "09:45"}
      ]
    }
  ]
}
```

#### 文本输入
在文本输入框中粘贴符合上述格式的JSON字符串。

### 2. 时间偏移设置

通过设置时间偏移（秒），可以调整铃声提醒的时机：
- 正数：提前响铃
- 负数：延后响铃

### 3. 使用时间偏移计算器

应用提供智能的时间偏移计算器，可根据学校的铃声准确度更改。

## 适用场景

- 需要严格遵守课程时间的学生
- 容易忘记上下课时间的用户
- 希望提高时间管理效率的学习者

## 注意事项

1. 课程表需按照指定的JSON格式导入
2. 时间格式采用24小时制（如："08:00"）
3. 周日对应数字为7（非0）
